Microsoft ASP.NET WebHooks GitHub Receiver

This sample illustrates how to wire up a GitHub WebHooks receiver. A sample WebHook URI is:

https://<host>/api/webhooks/incoming/github/{id}?code=83699ec7c1d794c0c780e49a5c72972590571fd8

Set the 'MS_WebHookReceiverSecret_GitHub' application setting to the application secret, optionally using IDs to differentiate between multiple WebHooks, for example 'secret0, id1=secret1, id2=secret2'.

Please see GitHub WebHooks for more information.
